# 📖 Miniguia de Estudos

## O que é Análise de Dados?

A análise de dados consiste no processo de coletar, organizar, limpar, interpretar e visualizar dados com o objetivo de gerar informações relevantes para apoiar a tomada de decisões.

Ela permite identificar padrões, tendências e oportunidades, auxiliando empresas na melhoria de processos e na obtenção de vantagem competitiva.

---

## Tipos de Análise de Dados

### Análise Descritiva

Responde à pergunta:

**O que aconteceu?**

Utiliza relatórios e dashboards para apresentar informações históricas.

### Análise Diagnóstica

Responde à pergunta:

**Por que aconteceu?**

Busca identificar causas e relações entre os dados.

### Análise Preditiva

Responde à pergunta:

**O que provavelmente acontecerá?**

Utiliza modelos estatísticos e dados históricos para realizar previsões.

### Análise Prescritiva

Responde à pergunta:

**O que deve ser feito?**

Sugere ações e estratégias com base nos resultados das análises.

---

## Etapas de um Projeto de Análise de Dados

1. Planejamento e definição dos objetivos.
2. Definição do problema de negócio.
3. Coleta dos dados.
4. Limpeza e tratamento dos dados.
5. Processamento e análise.
6. Visualização dos resultados.
7. Geração de insights.
8. Apoio à tomada de decisão.

## Principais desafios

* Garantir a qualidade dos dados.
* Integrar dados de diferentes fontes.
* Proteger informações sensíveis.
* Interpretar corretamente os resultados.
* Trabalhar com grandes volumes de dados (Big Data).

# 📚 Glossário

**Big Data:**
Grandes volumes de dados gerados continuamente.

**Business Intelligence (BI):**
Conjunto de ferramentas para análise e visualização de dados.

**Dashboard:**
Painel que apresenta indicadores e gráficos.

**Data Lake:**
Repositório que armazena dados brutos.

**Data Warehouse:**
Banco de dados estruturado para análises.

**ETL:**
Processo de Extrair, Transformar e Carregar dados.

**ELT:**
Processo de Extrair, Carregar e Transformar dados.

**Insight:**
Informação relevante obtida a partir da análise dos dados.

**KPI:**
Indicador-chave utilizado para medir desempenho.

**Machine Learning:**
Técnicas que permitem aos computadores aprenderem com os dados.
